# Personal-use synthetic fixtures

<section lang="fa" dir="rtl">

این fixtureها فقط برای demo کنترل‌شده و تست بسته‌بندی مصرف شخصی هستند.

قواعد:

- همگی synthetic هستند.
- نباید به‌عنوان handoff واقعی EV4 معرفی شوند.
- نباید ادعای real Elementor validation، frontend correctness، accessibility completion، export validation یا production readiness بسازند.
- اگر شواهد واقعی وجود ندارد، نتیجه نهایی demo باید `insufficient_evidence` باقی بماند.


برای demo کنترل‌شده مسیر پیش‌فرض `uv run python scripts/run-project-gate-demo.py` است.
